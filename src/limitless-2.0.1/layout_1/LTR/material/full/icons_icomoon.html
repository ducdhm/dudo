Icomoon icon set

Icomoon is a custom icon set, includes 1145 icon glyphs based on 16px grid. This set is a default icon set of the template, majority of components are using Icomoon font family for UI elements instead of images, so all of them are retina-ready. Material and FontAwesome icon sets are also added, but as optional choice. In order to get the best look of UI elements, I recommend to include this icon set only since you can find perfectly crafted icons according to most of your common needs.
