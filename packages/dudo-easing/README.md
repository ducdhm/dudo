# @dudojs/easing
Collection of awesome easing functions!

## Usage
```javascript
import easingList from '@dudojs/easing';
const duration = 500;
const easing = 'linear';

doSomeAnimate(duration, easingList[easing]);
```

## Available easing
 * linear
 * easeInQuad
 * easeOutQuad
 * easeInOutQuad
 * easeInCubic
 * easeOutCubic
 * easeInOutCubic
 * easeInQuart
 * easeOutQuart
 * easeInOutQuart
 * easeInQuint
 * easeOutQuint
 * easeInOutQuint
