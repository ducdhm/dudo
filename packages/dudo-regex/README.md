# @dudojs/regex
Collection of awesome regular expression!

## Usage
```javascript
import REGEX from '@dudojs/regex';

REGEX.EMAIL.test('ducdhm@gmail.com');
```

## Available Regex
 * linear
 * easeInQuad
 * easeOutQuad
 * easeInOutQuad
 * easeInCubic
 * easeOutCubic
 * easeInOutCubic
 * easeInQuart
 * easeOutQuart
 * easeInOutQuart
 * easeInQuint
 * easeOutQuint
 * easeInOutQuint
